# FocusForge 🔥

FocusForge is a modern, open-source productivity web app designed to help students and learners stay focused, track their study sessions, and build consistent habits.

Built with simplicity and performance in mind, FocusForge combines a clean UI with powerful features to support deep work and efficient study routines.

---

## ✨ Features

* ⏱ **Custom Focus Timer**
  Set your own study duration and stay in the zone.

* 📚 **Dynamic Subject Tracking**
  Add and manage your own subjects or goals.

* 📊 **Study Analytics**
  View session stats and track your progress over time.

* 💾 **Local Storage Support**
  Your data is stored locally — no login required.

* 🎨 **Modern UI**
  Clean, responsive, and distraction-free design.
