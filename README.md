# FocusForge 🔥

FocusForge is a modern productivity web app designed to help students stay focused and track their study sessions.

## Features
- ⏱ Focus Timer (custom duration)
- 📚 Dynamic Subject Tracking
- 📊 Study Statistics
- 💾 Data saved using localStorage

## Tech Stack
- HTML
- CSS
- JavaScript

## Future Improvements
- Break system
- Streak tracking
- Weekly analytics
